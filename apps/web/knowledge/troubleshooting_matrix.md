# 咖啡沖煮感官除錯矩陣 (Sensory Troubleshooting Matrix)

**[標籤：感官評估, 參數微調, 萃取理論, 除錯, Troubleshooting]**
本指南結合 SCA 金杯理論、流體力學與熱力學，提供針對負面感官體驗的具體參數調整建議。調整時應遵守「一次只變動一個變數 (One variable at a time)」的原則。

---

## 情境 A：萃取不足 (Under-extraction)
**[標籤：萃取不足, 酸澀, 尖銳, 缺乏甜感, 水感, 參數調整]**
* **感官特徵**：酸味過於尖銳刺鼻 (Sharp/Sour)、缺乏甜感支撐、餘韻短促、口腔感覺單薄 (水感 Watery)、甚至帶有未熟的青草味或穀物味。
* **物理原因**：水與咖啡粉的接觸時間不足，或動能不夠，導致中後段的糖分與醇厚度化合物尚未溶解出。
* ** 建議調整清單 (依優先級排列)**：
    1.  **調細研磨度 (Grind Finer)**：呈指數型增加咖啡粉的總表面積，大幅提升萃取速率。
    2.  **提高水溫 (Higher Temperature)**：增加熱力學動能，每次提升 1°C - 2°C，加速物質溶解。
    3.  **增加擾流 (More Agitation)**：在注水時提高水位落差，或增加繞圈速度，甚至使用湯匙輕微攪拌，打破粉層通道。
    4.  **拉長粉水比 (Higher Ratio)**：若原本為 1:15，可增加水量至 1:16，讓乾淨的水帶出更多剩餘物質。

## 情境 B：萃取過度 (Over-extraction)
**[標籤：萃取過度, 焦苦味, 乾澀, 咬喉嚨, 雜味, 參數調整]**
* **感官特徵**：出現強烈的焦苦味 (Bitter)、吞嚥後口腔感到乾燥粗糙 (澀感 Astringent)、風味混濁缺乏層次 (Muddy)、木質味或藥水味重。
* **物理原因**：萃取進行得太徹底，導致高分子量的苦味與澀味化合物（如綠原酸降解物、單寧酸）大量進入咖啡液中。
* ** 建議調整清單 (依優先級排列)**：
    1.  **調粗研磨度 (Grind Coarser)**：減少表面積，減緩水流阻力，加快流速以避免過度浸泡。
    2.  **降低水溫 (Lower Temperature)**：尤其針對中深焙咖啡豆，降低 2°C - 3°C 能有效抑制高分子量苦味物質的溶出。
    3.  **減少注水段數 (Fewer Pours)**：將四段注水改為兩段或一刀流，減少每次斷水重新注水時產生的擾流翻滾。
    4.  **縮短粉水比 (Lower Ratio)**：若原本為 1:16，可減少水量至 1:14 或 1:15，提早截斷尾段容易產生雜味的水流，若覺得濃度太高，可再額外加水稀釋 (Bypass)。
    5.  **更換快流速濾杯/濾紙**：改用錐形濾杯 (如 V60) 或快流速濾紙 (如 CAFEC 淺焙)，強行縮短總萃取時間。

## 情境 C：通道效應 (Channeling Effect)
**[標籤：通道效應, 混合風味, 澀感, 佈粉, 參數調整]**
* **感官特徵**：同時出現「尖銳的酸味」與「咬喉嚨的焦苦澀味」。這代表粉層中有一部分極度萃取不足，而另一部分極度萃取過度。
* **物理原因**：水流沒有均勻穿過所有咖啡粉，而是尋找阻力最小的「捷徑」快速流過，導致該通道內的粉被過度沖刷，而周圍的粉幾乎沒被萃取到。
* ** 建議調整清單**：
    1.  **檢查佈粉 (Bed Preparation)**：注水前務必輕拍濾杯，確保粉床平整。
    2.  **改善注水水柱 (Pouring Technique)**：注水過於猛烈會直接擊穿粉層。應保持水柱柔和垂直，均勻繞圈。
    3.  **篩除極細粉 (Sift Fines)**：若磨豆機品質不佳產生過多極細粉 (Fines)，容易堵塞底部導致水流亂竄。可考慮稍微調粗研磨度或升級磨豆機。
